# MedControl — Visão Geral e Objetivo

## O que é
Sistema SaaS de gestão de medicamentos para farmácias e redes farmacêuticas.
Permite controle de estoque, validade, alertas e gestão de múltiplas filiais.

## Público-alvo
- Farmácias independentes
- Redes de farmácias (múltiplas filiais)
- Proprietários que precisam de visão centralizada

## Problema que resolve
Farmácias perdem dinheiro com medicamentos vencidos e falta de controle de estoque.
O MedControl centraliza tudo em um painel web acessível de qualquer lugar.
