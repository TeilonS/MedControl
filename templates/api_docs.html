📋 Endpoints

GET/medicamentos
Lista todos os medicamentos da rede. Filtre por status ou filial via query string.

Parâmetros opcionais

ParâmetroTipoDescrição
statusstringFiltra por status: vencido, alerta_30, alerta_60, ok
filial_idintegerFiltra por filial específica
 
curl -H "X-API-Key: SUA_CHAVE" \
     https://www.medcontrol.app.br/api/v1/medicamentos

# Filtrar apenas vencidos:
curl -H "X-API-Key: SUA_CHAVE" \
     "https://www.medcontrol.app.br/api/v1/medicamentos?status=vencido"
import requests

resp = requests.get(
    "https://www.medcontrol.app.br/api/v1/medicamentos",
    headers={"X-API-Key": "SUA_CHAVE"},
    params={"status": "vencido"}  # opcional
)
print(resp.json())

POST/medicamentos
Cadastra um novo medicamento. Ideal para integração com ERP/Consys na entrada de estoque.

Campos do body (JSON)

CampoTipoDescrição
nomestringobrig.Nome do medicamento
lotestringobrig.Número do lote
data_validadestringobrig.Formato YYYY-MM-DD
quantidadeintegerobrig.Quantidade em estoque
codigo_barrasstringopcionalEAN-13 ou código interno
fabricantestringopcionalNome do fabricante
preco_unitariofloatopcionalPreço unitário em R$
filial_idintegeropcionalID da filial (se omitido, vincula à rede)
origemstringopcionalEx: consys, tasy, sngpc
codigo_externostringopcionalID do item no sistema de origem
 
curl -X POST \
     -H "X-API-Key: SUA_CHAVE" \
     -H "Content-Type: application/json" \
     -d '{"nome":"Amoxicilina 500mg","lote":"LT-001","data_validade":"2027-03-15","quantidade":50,"preco_unitario":8.90,"origem":"consys"}' \
     https://www.medcontrol.app.br/api/v1/medicamentos
import requests

resp = requests.post(
    "https://www.medcontrol.app.br/api/v1/medicamentos",
    headers={
        "X-API-Key": "SUA_CHAVE",
        "Content-Type": "application/json"
    },
    json={
        "nome": "Amoxicilina 500mg",
        "lote": "LT-001",
        "data_validade": "2027-03-15",
        "quantidade": 50,
        "preco_unitario": 8.90,
        "origem": "consys"
    }
)
print(resp.json())
# Sucesso (201)
{
  "success": true,
  "id": 42,
  "message": "Medicamento cadastrado com sucesso."
}

# Erro (400)
{
  "success": false,
  "error": "Campos obrigatorios: lote, data_validade"
}
PUT/medicamentos/{id}
Atualiza dados de um medicamento existente. Envie apenas os campos que deseja alterar.
curl -X PUT \
     -H "X-API-Key: SUA_CHAVE" \
     -H "Content-Type: application/json" \
     -d '{"quantidade": 80}' \
     https://www.medcontrol.app.br/api/v1/medicamentos/42
resp = requests.put(
    "https://www.medcontrol.app.br/api/v1/medicamentos/42",
    headers={"X-API-Key": "SUA_CHAVE"},
    json={"quantidade": 80}
)
print(resp.json())
DELETE/medicamentos/{id}
Remove um medicamento da base de dados.
curl -X DELETE \
     -H "X-API-Key: SUA_CHAVE" \
     https://www.medcontrol.app.br/api/v1/medicamentos/42

GET/medicamentos/barcode/{codigo}
Busca um medicamento pelo código de barras (EAN-13 ou código interno).
curl -H "X-API-Key: SUA_CHAVE" \
     https://www.medcontrol.app.br/api/v1/medicamentos/barcode/7891234567890

🏷️ Valores de Status

ValorSignificado
vencidoData de validade já passou
alerta_30Vence em até 30 dias
alerta_60Vence entre 31 e 60 dias
okVence em mais de 60 dias
